# Organizador de Estudos CLI

Este é um projeto simples em Python para organizar tarefas de estudo.

## Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Remover tarefas

## Como rodar o projeto

1. Instale o Python
2. Instale as dependências:

pip install -r requirements.txt

3. Execute o projeto:

python src/app.py

## Como rodar os testes

python -m pytest
